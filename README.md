--HIMARTIA--
A Godot 4 action-platformer inspired by Hollow Knight, Built for mobile.

| Overview
HIMARTIA is a 2D action-platformer currently in early development using Godot 4. Drawing heavy inspiration from Hollow Knight, the game explores themes of guilt, and descent from heroism to whatever the fuck.

This repo contains the core game logic, scripts, and assets for it.

| Tech Stack
- Engine: Godot 4
- Target Platform: Android
- Graphics: Pixel art
- Version Control: GitHub

| Features (WIP)
- Modular movement system (dash, wall jump, wall sliding, Double Jump)
- Power-Ups systems named "RELIC STONES"
- Biome-based level design with layered parallax
- Early biome tilesets in progress  
- UI mockups and animation logic under refinement
